The function of the app
	This app is used to generate grocery shopping lists based on the recipies that the user would like to make. 
The storyline
	The user authenicates to our server.
	The user then searches for recipies that they would like to make.
	Based on the recipies users can add items to their Kroger cart. 
Dependencies to run the apps and any API-related requirements
	Users need a Kroger Account in order to transfer item from our app to Kroger and be able to checkout with their items.
